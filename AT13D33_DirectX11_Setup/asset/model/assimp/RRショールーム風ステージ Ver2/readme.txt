AutoLuminous対応です
地面影は非表示にして使用してください

"鏡面床モデル"フォルダ内のxファイルはWorkinfFloor系MME用の床を鏡面反射にするためのアクセサリです。
使用する場合はWorkinfFloor等のエフェクトフォルダに置いて、fxファイルとxファイルの名前を同じにして下さい。
MMDで読み込んだら"床.x"は床ボーンに、"台座.x"は台座ボーンを親に設定することをおすすめします。

monitor.xを読み込むと液晶モニターに背景aviが流せます。
